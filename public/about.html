About This Project
This selfbot allows users to update their Discord rich presence (RPC) easily through a web interface. It includes various commands for fun, utilities, and Discord-related features.

Commands & Features!
  • Fun Commands: Fake quote, random avatars, user search, etc.
  • Utility Commands: Server info, guild icon fetcher, message lookup, etc.
  • RPC Maker: Set custom statuses like "Streaming," "Playing," and more.

And many more commands!
